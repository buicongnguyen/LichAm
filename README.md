# Lich Am

Solar and Vietnamese-style lunar calendar web app with memory days, holiday markers, and phone-calendar export.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This repo includes a GitHub Actions workflow at `.github/workflows/deploy.yml`. On every push to `main`, it builds the app and deploys `dist` to GitHub Pages.

For a project repo named `LichAm`, the Pages URL will be:

```text
https://buicongnguyen.github.io/LichAm/
```

For a user-site repo named `buicongnguyen.github.io`, the Pages URL will be:

```text
https://buicongnguyen.github.io/
```
